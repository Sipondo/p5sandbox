## ğŸ’¡ Learning Path

For beginners, it's recommended to go through the sketches in this order:
1. Start with **Basics** to understand the canvas
2. Move to **Shapes** to learn drawing
3. Explore **Colors** to add visual variety
4. Try **Loops** to create patterns efficiently
5. Learn **Conditionals** for decision making
6. Experiment with **Animation** to make things move
7. Add **Interaction** for user input
8. Use **Arrays** to manage multiple objects
9. Create **Functions** for reusable code

All sketches can be run directly in a web browser. No installation or build process is needed. Simply open any `.html` file in your browser.

You can also serve the files using a local server:
```bash

npx http-server
```

Then navigate to `http://localhost:8000` in your browser.
